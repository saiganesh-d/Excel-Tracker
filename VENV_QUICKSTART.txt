================================================================================
                           HOW TO BUILD EXE
================================================================================

OPTION 1: EASIEST (Auto-detects your venv)
------------------------------------------

OPTION 2: MANUAL (If you know your venv name)
----------------------------------------------

build_exe.bat

================================================================================
                         TROUBLESHOOTING
================================================================================

ERROR: "Python is not recognized"
SOLUTION: Activate venv first → venv\Scripts\activate.bat

ERROR: "PyInstaller not found"
SOLUTION: pip install pyinstaller

ERROR: "Cannot find activate.bat"
SOLUTION: Check your venv folder name (might be .venv or env)

ERROR: PowerShell script execution disabled
SOLUTION: Use Command Prompt instead → venv\Scripts\activate.bat

================================================================================
                         YOUR WORKFLOW
================================================================================

DAILY DEVELOPMENT:
1. Open terminal in project folder
2. Activate: venv\Scripts\activate.bat
3. Run: streamlit run app.py
4. Work on your files
5. Deactivate when done: deactivate

BUILDING EXE FOR DISTRIBUTION:
1. Run: build_with_venv.bat
2. Wait 5-10 minutes
3. Find EXE at: dist\DocumentComparison\DocumentComparison.exe
4. Zip the DocumentComparison folder
5. Share with users

INSTALLING NEW PACKAGES:
1. Activate: venv\Scripts\activate.bat
2. Install: pip install package_name
3. Update: pip freeze > requirements.txt
